# Vue-covid-dashboard

A dummy web application written on **Vue.js 3** that allows users to keep track of dynamics of COVID-19 in a chosen country. It contains a chart and a card with general information on statistics of the chosen country. Global statistics are also available, but not as a chart.
Used technologies and APIs:

 - Vue-cil;
 - Chart.js library (not integrated with Vue.js);
 - [COVID-19 API](https://about-corona.net/documentation);
 - [Country flags API](https://www.countryflags.io/).
 
 [Link to the app](https://kochtaik.github.io/vue-covid-dashboard/).
